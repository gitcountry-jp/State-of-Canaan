# カナアン島共和国 憲法 (Constitution)

本ドキュメントは、カナアン島共和国（State of Canaan）における最高法規であり、国家のシステムおよび運営ルールを定義する。

## 第一章：国家の定義
* 本国は、GitHubおよびDiscordを基盤とするデジタル民主主義国家である。
* 主権は国民（リポジトリのコントリビューターおよびDiscordの参加者）に属する。

## 第二章：立法（法律の制定）
* すべての法律の提案は、GitHubの **Issue** または **Pull Request (PR)** によって行われなければならない。
* 提出された法案（PR）は、国民による議論と合意形成を経て、マージ権限を持つ者によってリポジトリへ結合（Merge）されることで即時発効する。

## 第三章：参加者の権利と義務
* すべての国民は、建設的な議論を行う権利を有し、コミュニティの治安を乱す行為（荒らしなど）をしてはならない。
