# カナアン島共和国

「政治をオープンソースにしたらどうなるか？」を試す実験場です。
GitHubのIssueで議論して、PR（プルリク）で法律をマージする、ブラウザ駆動型の仮想国家を作ります。

## なんで作ろうと思ったか
今の世の中の政治はブラックボックスだし、手続きもめんどくさい。
だったら、エンジニアが普段やってる「Gitの文化」をそのまま国に持ち込んだ方が、圧倒的にクリーンで爆速な民主主義ができるんじゃないか？と思ったのがきっかけです。

ターミナルを使わずに、ブラウザのGitHub UIだけで完結する運営を目指すので、非エンジニアの人も巻き込んでワイワイやりたいです。
Discordでもやってるよ！！！

## 法制度とルール（Documents）

カナアン島共和国の根幹となるルールです。専門的なファイルは以下に格納されています。

* **[憲法（constitution.md）](./docs/constitution.md)**

* **[法律集（laws/）](./docs/laws/)**
